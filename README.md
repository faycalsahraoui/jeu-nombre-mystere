# Trouve le nombre exact

Le but de ce jeu consiste à trouver le nombre exact entre 1 et 1000

# Utilisation

- Ouvrez le fichier `ind.html` sur le navigateur de votre choix
- Entrez un nombre au hasard puis appuyez sur la touche ENTREE
- **Trois réponses possibles peuvent s'afficher** ;
- 1/ **Trop Petit** : le nombre que vous avez entrée est trop petit, il faut entrée un nombre plus grand
- 2/ **Trop Grand** : le nombre que vous avez entrée est trop grand, il faut entrée un nombre plus petit
- 3/ **Bravo vous avez gagné** : le nombre que vous avez entrée est correcte, vous pouvez relancez une partie directement

# Structure du projet 

- `ind.html` : Structure principale de la page web 
- `style.css`: Apporte un style pour le fond de la page web
- `script` : Le code qui m'a permis de créer le jeu
- `fond css.jpg`: Ajout d'une image de fond pour la page web qui rend la page plus esthétique

# Technologies Utilisés 
- **HTML** : Structure de la page
- **CSS** : Mise en page et design
- **JavaScipt** : Langage de programmation pour le code du jeu

# Amélioration possible

Le jeu de base est simple et efficace mais j'aurai aimé faire des améliorations dessus mais par manque de temps je suis restée dans la simplicité. Cependant pour les mises à jours à venir il pourrait avoir comme nouveautés :
- Un système de chances pour trouver le nombre (peut-être 8 chances pour trouver le nombre sinon tu perd!)
- Un ajustement des messages pour être plus précis (ex : si le mnombre est 500 et que l'utilisateur écrit 400 le message affiché est petit au lieu de trop petit)
